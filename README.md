## üöÄ Key Features

- **Fully Responsive Design**: Adapts seamlessly to all devices, from mobile phones to desktops.
- **Dark & Light Mode**: A user-friendly toggle to switch between themes, with the user's system preference detected on first visit.
- **Smooth Scroll Animations**: Subtle, professional animations on scroll reveal that enhance the user experience without being distracting.
- **Interactive UI Elements**: Modern buttons, hover effects, and a sticky navigation bar create an engaging interface.
- **Dynamic Content**: Project and skill sections are dynamically generated using JavaScript, making updates easy and efficient.
- **Clean & Modern UI**: A completely new design with a fresh color palette and professional typography.
